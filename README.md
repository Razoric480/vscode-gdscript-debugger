# Godot Debugger

Connects to the Godot remote debugger to give debugging capabilities to GDScript based Godot projects in VSCode.

## Features

- Launches the remote Godot debugger and parses its messages

## Requirements

- `godot` should be on the PATH

## Extension Settings

- godot_debugger.port_number: The port number to use to connect to the remote debugger.
